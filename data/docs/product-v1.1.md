# Product Docs (v1.1)

## Supported features
- Collections
- Basic queries
- Indexes (basic)
- Feature X (new)

## Feature X
Feature X is supported starting in v1.1.

Enable it by setting:
- `FEATURE_X_ENABLED=true`

## TLS
To enable TLS, set:
- `TLS_ENABLED=true`
- Provide certificates via `TLS_CERT_PATH` and `TLS_KEY_PATH`.
